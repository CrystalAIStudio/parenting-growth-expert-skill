# 使用说明

## 这个 Skill 适合在什么时候用

当你希望得到一个更成熟、更温和、更有方法的育儿视角时，可以调用这个 Skill。

它尤其适合：

- 青春期亲子冲突
- 孩子情绪问题
- 规则和边界建立
- 屏幕与学习问题
- 父母情绪管理
- 家庭环境长期改善

## 如何调用

不同客户端的调用方式略有不同，常见方式包括：

```text
/parenting_growth_expert
```

或者直接在对话中说：

```text
请用育儿成长专家的方式帮我分析这个问题。
```

## 推荐提问方式

### 1. 直接问具体问题

```text
我家孩子青春期以后特别顶嘴，不愿意和我沟通，我现在最该先做什么？
```

### 2. 问一个长期改善方向

```text
我们家现在经常因为作业和手机冲突，我想知道应该先从哪里调整家庭环境。
```

### 3. 让它给出咨询式回答

```text
请不要讲太多大道理，像一个有经验的育儿专家一样，先接住我的情绪，再告诉我最重要的一步该做什么。
```

### 4. 让它做系统方案

```text
请从关系、情绪、边界、习惯、环境五个层面，帮我设计一个未来两周的家庭调整方案。
```

## 最佳使用建议

- 尽量告诉它孩子的年龄段
- 尽量描述发生问题的具体场景
- 可以告诉它你最困扰的点是什么
- 如果你只想先做一件事，可以明确说“先只告诉我最重要的一步”
- 如果你想要更有对话感，可以让它“一次只说一个重点”

## 示例

### 用户

```text
我家孩子一写作业就拖拉，我一催他就发脾气，我也越来越没有耐心了。
```

### 期望表现

这个 Skill 应该：

- 先接住家长的消耗感
- 判断这不只是态度问题，也可能是任务难度、胜任感和亲子互动在叠加
- 先给一个最重要的动作
- 给一句可直接使用的话术
- 用一个轻量问题继续推进对话
